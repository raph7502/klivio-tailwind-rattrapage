🚀 Intégration Tailwind CSS - Projet Web
Projet d'intégration web complet (Desktop & Mobile) utilisant une approche Utility-First pour un rendu moderne et performant.

🔗 Lien du site : https://walidsooo.github.io/Integration-WEB-HTML-Tailwind_CSS-/

🛠️ Technologies
HTML5

Tailwind CSS (Framework utilitaire)

JavaScript (Interactions dynamiques)

Responsive Design (Optimisé pour tous les écrans)

📂 Structure
/assets : Images, icônes et ressources graphiques.

/src : Fichiers sources et configuration des styles.

/ressources : Maquettes et documents de référence.

index.html : Page principale du projet.

tailwind.config.js : Personnalisation du framework.
